# Agent Telemetry SDK (v2.0.0)

Structured, secure, multi-framework telemetry and monitoring SDK for Python applications.

This SDK provides enterprise-grade monitoring capabilities including:

- Exception tracking
- Incoming & outgoing HTTP monitoring
- Database query monitoring
- Structured logging capture
- Performance tracking
- HMAC-secured transport
- Retry with exponential backoff
- AI-ready structured event schema
